# swsec-intro-php
PHP port for [SWSEC-INTRO](https://github.com/injcristianrojas/swsec-intro).
Will document this soon.

# Requirements

* PHP-CLI
* PHP-PDO (sqlite3)

# Usage

Start the server using

```shell
php -S localhost:8000 -t app/ -c php.ini
```

Then go to http://localhost:8000 and have fun.
